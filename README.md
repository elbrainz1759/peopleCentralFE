# HR Notification System

A specialized HR Management Dashboard designed for tracking employee contracts, leaves, and exits. This system provides a streamlined, minimalist interface for HR administrators to proactively manage staff records and upcoming contract renewals.

## 🚀 Key Features

### 📅 Notification Tracker
- **Proactive Contract Monitoring**: Automatically identifies contracts expiring within 2 months.
- **Expiry Logic**: Real-time calculation of days remaining until contract end.
- **Staff Notifications**: Quick-action prompts to notify staff or managers about upcoming renewals.

### 👥 Employee Database
- **Staff Directory**: Centralized view of all staff with integrated search and filtering.
- **Contract Tracking**: Integrated contract start and end dates for every staff member.
- **Detailed Profiles**: Comprehensive profile views including professional, contact, and contract information.

### 🌴 Leave Management
- **Request Workflow**: Multi-step leave application process.
- **Approval System**: Dedicated dashboard for managers to review and approve leave requests.
- **Reporting**: Detailed leave reports and staff history logs.

### 🚪 Exit Management
- **Structured Offboarding**: Standardized exit checklists and handover tracking.
- **Exit Interviews**: Digital capture of feedback and offboarding documentation.
- **Multi-stage Approvals**: Formalized approval workflow for staff exits.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS V4
- **Language**: TypeScript
- **State Management**: React Context API
- **Icons**: Custom SVG System

## 📦 Getting Started

### Prerequisites

- Node.js 20.x or later
- pnpm

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Pascal-Ahmadu/HRNotificationSystem.git
    cd HRNotificationSystem
    ```

2.  **Install dependencies**:
    ```bash
    pnpm install
    ```

3.  **Start the development server**:
    ```bash
    pnpm dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🎨 UI & Design Principles

The system features a **minimalist, distraction-free interface** designed for professional efficiency:
- **Focused Header**: Removed redundant search and utility icons to focus on the system title.
- **Clean Sidebar**: Stripped of generic template noise, showing only relevant HR modules.
- **Responsive Tables**: Optimized for clarity and at-a-glance monitoring.
